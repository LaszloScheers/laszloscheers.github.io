IBAT College Diploma in Web Design

The IBAT college diploma in web design is a ten-week learning journey that takes you from the basics of working with HTML5 and CSS3 files to building a personal portfolio website.

    Course content:

  • Introduction to HTML, CSS and hosting on GitHub.
  • Working with VS Code, colours, images and fonts.
  • Working with Google Analytics, the box model.
  • Introduction to responsive web design and CSS flexbox.
  • Hero blocks, contact forms, menus and footers.
  • Building your personal portfolio website.
